🌿 AAUAMart
A mobile-first marketplace connecting farmers and consumers at Adekunle Ajasin University, Akungba-Akoko (AAUA).
AAUAMart is a full-stack React + Ionic application backed by Firebase that lets buyers shop for fresh agricultural produce, sellers list and manage their products, and admins oversee the entire platform — all from a single codebase that runs on web, Android, and iOS.

📸 App Overview
Role
What they can do
🛒 Buyer
Browse products, add to cart, place orders, track delivery
🏪 Seller
List products with images, manage inventory, update order status
🛡️ Admin
View all users, all orders, platform revenue stats
